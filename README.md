# PimpMyMatrix
Matrix display with ESP

# Purpose

 - Display text on a 32x8 matrix.
 - Display shapes on a 32x8 matrix.

# Connection

 - Wifi
 
# Features

 - Web server embbeded
 - Scrolling mode (Speed tunable)
 - Embedded Editor page
